📦 Backup Manuale

Esporta o importa i dati come file JSON (utile come backup o su browser che non supportano File System API).

⚠️ Zona Pericolosa

Questa azione eliminerà TUTTI i dati salvati in modo irreversibile.

ℹ️ Come funziona la sincronizzazione

Prima volta (uno solo):
1. Clicca Crea nuovo file JSON → scegli la cartella Google Drive condivisa → dai un nome (es. finanze_famigliari.json)

L'altra persona:
2. Aprire l'app → Impostazioni → Apri file esistente → seleziona lo stesso file dalla cartella Drive

Uso quotidiano:
3. Ogni modifica scrive automaticamente sul file → Drive sincronizza → l'altro vede i dati aggiornati! Usa 🔄 per ricaricare i dati se l'altro ha fatto modifiche.
